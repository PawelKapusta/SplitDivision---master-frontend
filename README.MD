# SplitDivision---master-frontend

## Available Scripts

### Running the development server.

### To install
```bash
    npm install
```
### And run:

```bash
    npm run dev
```

### Building for production.

```bash
    npm run build
```

### To run project

#### Build docker image

```bash
    docker build -t master-frontend .
```

#### Run a container on port 3000

```bash
    docker run -p 3000:3000 master-frontend
```
